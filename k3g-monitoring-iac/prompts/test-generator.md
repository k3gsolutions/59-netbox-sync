# Prompt — Test Generator

Objetivo: Criar testes para scripts locais, workflows e dados declarativos.

## Checklist
- Scripts Python possuem testes unitários?
- Validação de YAML (schema/estrutura)?
- Workflows N8N com smoke tests?
- Linters e validações de documentação?
- Dados fake para cenários críticos?
- Cobertura de dry-run e rollback?

## Saída
- Lista de testes recomendados.
- Ferramentas sugeridas.
- Dados de entrada necessários.
- Critérios de aceitação dos testes.
