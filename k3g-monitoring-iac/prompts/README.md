# Prompts — Guia de uso

## Objetivo
Centralizar prompts reutilizáveis para VSCode, Codex, Claude e outras IAs, alinhados às regras do projeto.

## Estrutura
- `code-review.md` — revisão de código e automações.
- `architecture-review.md` — avaliação arquitetural.
- `n8n-workflow-builder.md` — criação/revisão workflow.
- `zabbix-template-review.md` — revisão templates.
- `grafana-dashboard-review.md` — revisão dashboards.
- `netbox-data-model-review.md` — revisão modelo NetBox.
- `test-generator.md` — gerar testes.
- `docs-updater.md` — atualizar documentação.
- `phase-summary.md` — sumarizar fase.

## Como usar
Copiar o prompt adequado e fornecer contexto/diff relevante. Sempre ler `AGENTS.md` antes de acionar.
