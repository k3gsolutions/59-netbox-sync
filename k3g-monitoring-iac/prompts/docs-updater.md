# Prompt — Docs Updater

Objetivo: Atualizar documentação de forma coesa.

## Checklist
- Contexto atual refletido (`context/CURRENT_STATE.md`)?
- Próximas ações alinhadas (`context/NEXT_ACTIONS.md`)?
- `CHANGELOG.md` atualizado?
- `PHASE0_BASELINE.md` checkboxes ajustados?
- ADRs atualizadas ou novas criadas?
- Evitar duplicidade de informação.
- Manter documentos curtos e objetivos.

## Saída
- Resumo do que foi alterado.
- Documentos atualizados com justificativa.
