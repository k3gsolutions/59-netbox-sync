# 02 — Fase 0 Baseline

## Objetivo
Organizar documentação, contexto operacional, prompts, skills, ferramentas locais, taxonomias iniciais e estrutura GitOps para suportar fases subsequentes.

## Entregáveis
- Estrutura de diretórios e documentação.
- Checklists e roadmap faseado.
- ADRs iniciais.
- Prompts/skills reutilizáveis.
- Scripts locais stub.
- Taxonomias iniciais de NetBox e Zabbix.

## Critérios de aceite
- IA consegue trabalhar com contexto curto.
- Documentação orienta frentes A e B.
- Ferramentas locais ajudam a manter contexto atualizado.
- Sem automação real aplicada.

## Dependências
Nenhuma. Serve de fundação para as próximas fases.
