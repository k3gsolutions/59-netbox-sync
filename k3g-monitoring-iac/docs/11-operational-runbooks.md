# 11 — Operational Runbooks

## Objetivo
Garantir que o time (incluindo júnior) consiga operar a plataforma com procedimentos claros.

## Estrutura sugerida
- Onboarding de dispositivos/circuitos.
- Resposta a incidentes de automação.
- Reprocessamento de DLQ.
- Atualização de templates/dashboards via GitOps.
- Auditoria recorrente com `netops_netbox_sync`.
