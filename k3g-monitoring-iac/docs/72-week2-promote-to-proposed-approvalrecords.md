# Promoção para ApprovalRecords Propostos

A promoção da Semana 2 só roda depois de decisão humana válida.

## Regras
- `approve_for_approval_record`
- `reviewer` preenchido
- `reviewed_at` preenchido
- `approval_record_allowed=true`
- `reason` ou `notes` preenchido

## Garantias
- Saída fica em `approvals/pending/`
- Status fica `proposed/pending`
- Nenhum NetBox write
- Nenhum ApplyPlan
